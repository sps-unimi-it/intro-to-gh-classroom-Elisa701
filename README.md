# C4CSS exam, Vassallo — Replication study: sentiment dictionaries on BBC News

Repository for the exam project of the **c4ss** (Computational Social Science) course.  
The study replicates the approach of Chan et al. (2021) and the reference notebook  `Sentiment_analysis.ipynb`, applying three off‑the‑shelf sentiment lexicons to a different corpus: **9 recent short BBC News articles**, three for each category: *sport*, *technology*, *health*.

## Project structure

The folder contains the BBC News corpus (9 .txt articles), divided into three categories (health, sport, tech), the main notebook with the analysis, a local copy of the three sentiment lexicons, and the results generated by the notebook (CSV files and figures).

Each `.txt` file includes a `Title:` line, a `URL:` line and  and the body of the article, following the structure of `Data/Presidents_texts/`.  
Files are organized into subfolders by category. (`health/`, `sport/`, `tech/`).

## Lexicons used

The three off‑the‑shelf dictionaries required for the exam are:

1. **AFINN-111** —  integer scores from −5 to +5 for each word.
2. **Bing Liu** — lists of positive and negative words.
3. **NRC Emotion Lexicon** —  associations between words and eight emotions, including `positive` and `negative`.

The notebook looks for the lexicons in `../downloaded_lexica/` (as in the original course structure). If that folder is not available, it automatically uses the local copy in `Esame_Elisa/lexica/`.

## How to run the analysis with a virtual environment (venv)

1. Open the **Terminal**.
2. Move to the project folder:

   ```bash
   cd /Users/davides/Desktop/Esame/Esame_Elisa
   ```

3. Create a virtual environment:

   ```bash
   python3 -m venv .venv
   ```

4. Activate it:

   ```bash
   source .venv/bin/activate
   ```

   The terminal will show you `(.venv)` at the beginning of the line.

5. Install the required libraries:

   ```bash
   python3 -m pip install pandas numpy matplotlib scipy jupyter
   ```

6. Verify the installation:

   ```bash
   python3 -c "import pandas, numpy, matplotlib, scipy, jupyter; print('Tutto OK')"
   ```

7. Launch Jupyter:

   ```bash
   jupyter notebook Esame_Elisa_BBC.ipynb
   ```

8. Run all the cells: in the notebook's menu select `Cell → Run All`.
9. The CSV files and figures will be saved automatically in `Esame_Elisa/Results/`.


### Deactivate the environment

When finished, write in the terminal:.

```bash
deactivate
```

## Notes

- The articles were downloaded from official BBC URLs and saved as  `.txt` files in the subfolders of `Esame_Elisa/Data/BBC_news/`.
- The analysis uses  **length‑normalized scores** (`score / word_count`)  to replicate the correction for content‑length bias discussed in the reference study.
- Sport is the most positive category on average; health the most negative; technology is neutral or slightly positive depending on the lexicon.
